NGS Foundations · Reference

Glossary

The canonical vocabulary for this course. Terms defined here are used consistently in every lesson. Grouped roughly from molecule → run → files.

The library molecule

Insert (fragment)
The piece of sample DNA/cDNA you actually want to read. Everything else on the molecule exists to get this onto the flow cell and read it.
Adapter
Synthetic DNA ligated (or added by tagmentation) to both ends of the insert. Carries the flow-cell binding sites, index barcodes, and sequencing-primer sites.
P5 / P7
The two universal oligo sequences at the outermost ends of the molecule. They hybridize to complementary lawn oligos grafted on the flow cell — this is how a molecule attaches and seeds a cluster. P5 sits at the Read 1 end, P7 at the Read 2 end.
i7 index (Index 1, I1)
Sample barcode nearer the P7 end. Read during the Index 1 read.
i5 index (Index 2, I2)
Sample barcode nearer the P5 end. Read during the Index 2 read. Its reported orientation flips between instrument workflows — the classic reverse-complement sample-sheet gotcha.
Sequencing primer site (SP1 / SP2, Read 1 / Read 2 primer)
Where the sequencing primer anneals so synthesis (reading) can begin. SP1 launches Read 1 into the insert from the P5 side; SP2 launches Read 2 from the P7 side.
Index / barcode
A short known sequence (typically 8–10 bp) identifying which sample a molecule came from. Enables multiplexing.
UMI (unique molecular identifier)
A random barcode tagging each original molecule before amplification, so PCR duplicates can be collapsed. Distinct from a sample index.

The run

Denaturation
Splitting the double-stranded library into single strands (NaOH, pH >12.5) just before loading. Required because the flow-cell lawn can only hybridize single-stranded template. Incomplete denaturation → low cluster density.
Linearization
After clustering, cleaving and washing away one strand orientation so every strand in a cluster is single-stranded and points the same way — ready for the sequencing primer.
Cluster
A localized clonal copy-cluster of one library molecule on the flow cell, bright enough to image. One cluster → one read pair.
SBS (sequencing by synthesis)
Illumina's chemistry: add one fluorescent reversible-terminator nucleotide per cycle, image the color, then unblock. One cycle = one base.
Read 1 / Read 2
The two ends of the insert that get sequenced in paired-end mode. Each is a separate set of SBS cycles. Single-end omits Read 2.
Read sequence order
On one cluster, in one run: Read 1 → Index 1 (i7) → Index 2 (i5) → paired-end turnaround → Read 2. Each read is launched by annealing its own primer, then the new strand is stripped before the next.
Paired-end turnaround (resynthesis)
Mid-run step that rebuilds the complementary strand (bridge + copy) and removes the original Read 1 template, flipping the cluster's orientation so the Read 2 primer site is exposed. This is how Read 2 is possible after linearization; also the root of the i5 forward vs reverse-complement difference (i5 read before vs after the turnaround).
Read length
Number of SBS cycles per read, e.g. 2×150 = paired-end, 150 bp each end.
Multiplexing
Pooling many indexed samples into one lane/flow cell, then separating them computationally afterward (demultiplexing).
Patterned vs non-patterned flow cell
Patterned flow cells (NovaSeq, NextSeq, iSeq) have fixed nanowells and are prone to index hopping; non-patterned (MiSeq, HiSeq 2500) are not.
Coverage / depth
How many times, on average, each target base is read. Depth at a base; coverage over a region. See the coverage lesson for the math.
Bridge amplification / ExAmp
The two ways a single molecule is copied into a cluster. Bridge amplification (non-patterned) arches the strand to prime off neighboring lawn oligos; ExAmp (patterned) is a fast isothermal reaction whose excluded-volume effect keeps each nanowell monoclonal.
Nanowell
An etched well on a patterned flow cell that holds exactly one cluster at a fixed position.
Channel / chemistry (1- / 2- / 4-channel)
How many images per cycle an instrument takes to call a base. 2-channel (NextSeq/NovaSeq) encodes A=both, C=red, T=green, G=no signal.
Phasing / pre-phasing
Strands within a cluster falling behind (phasing) or running ahead (pre-phasing) of the cycle, so the cluster de-synchronizes. Accumulates over cycles → the downward slope in per-cycle quality plots.
Poly-G tail
A 2-channel artifact: dead/empty clusters emit no signal, which the caller reads as G. Shows up as runs of G at read 3′ ends.

Demux & files

Demultiplexing
Reading each cluster's index reads, matching them to the sample sheet, and routing the read into that sample's FASTQ. Illumina's tools: bcl2fastq / BCL Convert.
BCL
The raw per-cycle base-call files the instrument writes. Demux converts BCL → FASTQ.
Sample sheet
The table mapping each sample to its expected i7 (and i5) index. The input that tells demux how to split the pool.
Undetermined
Reads whose index didn't match any sample sheet entry within the allowed mismatch. A high Undetermined fraction is a first-line diagnostic.
Index hopping
A read gets mis-assigned because a free-floating index primed the wrong molecule. Mitigated by unique dual indexing (UDI).
Combinatorial vs unique dual indexing (CDI / UDI)
CDI reuses i7 and i5 pools in combinations; UDI pairs each i7 with one dedicated i5 so a hopped read can't masquerade as another real sample.
Index mismatch tolerance
How many base mismatches demux allows between an observed index and a sample sheet index and still assign the read. Default 1 per index. Too high → ambiguous assignment.
Index diversity / Hamming distance
Indexes in a pool must differ from each other by enough bases (edit distance) that a 1-base sequencing error can't turn one into another. Low diversity → misassignment and base-calling trouble; often mitigated with a PhiX spike-in.
i5 forward vs reverse-complement workflow
Instrument families read i5 off different strands, so the same index is reported as-is (MiSeq, HiSeq 2000/2500) or reverse-complemented (iSeq, NextSeq, NovaSeq, HiSeq 3000/4000/X). Root cause: whether i5 is read before or after the paired-end turnaround.
Adapter read-through
When the insert is shorter than the read length, sequencing runs off the end of the insert into the adapter — the bases you trim before analysis.

Library prep & sample types

Library prep
The bench workflow converting raw sample nucleic acid into adapter-flanked, sequenceable molecules. Always answers: get the nucleic acid, size it, attach adapters.
Ligation-based prep (TruSeq)
Fragment → end-repair + A-tailing → ligate adapters → PCR. Robust, wide input range, low bias.
Tagmentation (Nextera / Tn5)
A Tn5 transposase fragments DNA and inserts adapters in one step. Fast, low input, more bias. Source of the distinct "Nextera" adapter sequences you trim.
PolyA selection / rRNA depletion
The two RNA enrichment strategies. PolyA (oligo-dT) captures mRNA; rRNA depletion removes ribosomal RNA (~80–90% of total RNA) to keep everything else. Do one, or rRNA swamps the library.
Reverse transcription / cDNA
Copying RNA into DNA (cDNA) so it can be sequenced. First strand is single-stranded; second- strand synthesis makes it double-stranded, after which it's a standard DNA library.
Strandedness
Whether a library records which strand an RNA came from. Preserved by building the 2nd cDNA strand with dUTP and destroying it before PCR. Drives the strandedness flag in counting tools.
Hybrid capture
Enriching for target regions (exome/panel) by hybridizing the library to biotinylated probes and pulling them down. A WGS library plus a capture step.
Bulk vs single-cell
Bulk: one library = one sample, averaged over all cells; identity = sample index. Single-cell: one library holds thousands of cells, each tagged with a cell barcode + UMI inside the read.
Cell barcode
A per-cell tag (16 bp in 10x 3′ v3) shared by all molecules from one cell, drawn from a fixed whitelist. Lives in R1. Resolved by Cell Ranger — demux one level deeper.
Whitelist
The list of all valid cell barcodes (~3.5M for 10x). Functions like a sample sheet for cell barcodes; observed barcodes are matched to it within a mismatch budget.
Doublet / ambient RNA
Two cells sharing one barcode (doublet); free-floating RNA tagged with random barcodes (ambient) that smears background into every cell.
