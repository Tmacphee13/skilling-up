# Misconception corrected: ds-vs-ss is a workflow stage, not a molecule segment

After Lesson 2, Tanner asked how a double-stranded cDNA/insert can bind the flow cell, and
which *parts* of the molecule are ss vs ds. Underlying misconception: that single-strandedness
is a fixed property of certain segments. Corrected to: the whole finished library is
double-stranded in the tube; **denaturation (NaOH) before loading** makes it single-stranded;
binding is inherently ss (a single strand's P5/P7 adapter end hybridizes to a lawn oligo);
clustering oscillates ds↔ss; linearization leaves ss template for SBS.

**Implication:** He's now primed for library prep (L4) — he intuits the molecule as a physical,
strand-resolved object and asks mechanistic "how does this physically work" questions. Lean into
strand-level and step-level mechanism in future lessons; he engages with it. The denaturation →
low-cluster-density link is a good QC hook to reuse. Prior knowledge shown: understands
complementary strands, hybridization, and that the insert is ds. See [[MISSION.md]].
