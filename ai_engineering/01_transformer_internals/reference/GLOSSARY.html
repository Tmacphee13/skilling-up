Transformer Internals · Reference

Glossary

The controlled vocabulary for this course. Every lesson uses these terms as defined here. Grows as we go — terms are tagged with the lesson that introduced them.

Token
The atomic unit a language model reads and writes — not a word and not a character, but a chunk of bytes (often a common word-piece) chosen by the tokenizer. A model's entire world is a sequence of token IDs. Lesson 1
Vocabulary
The fixed set of all tokens a model knows (e.g. ~100k for GPT-4o). Each entry has an integer ID. Text outside the vocabulary is spelled out from smaller sub-word pieces. Lesson 1
BPE (Byte-Pair Encoding)
The algorithm that builds the vocabulary: start from raw bytes, then repeatedly merge the most frequent adjacent pair into a new token. Frequent words become single tokens; rare words stay in pieces. Lesson 1
Tokenization
The reversible process of converting text ↔ a sequence of token IDs, done before the model sees anything and after it produces anything. Lesson 1
Context window
The maximum number of tokens (not words or characters) a model can attend to at once. Cost and truncation are both counted in tokens. Lesson 1
Embedding
A vector of numbers standing in for a piece of language, so a network can do math on it. The word is overloaded — see the three types below. Lesson 2
Token embedding
A static vector looked up from the model's embedding table (one row per vocabulary entry, shape [vocab × d_model]), learned by backprop. Context-free: "bank" gets one row regardless of meaning. Lesson 2
Contextual embedding
A token's vector after passing through the transformer layers, reshaped by its neighbors via attention. "bank" now differs by sentence. The model's internal output. Lesson 2
Sentence embedding
One vector for a whole sentence/document, usually pooled (e.g. averaged) from contextual embeddings. What an embeddings API returns; stored for semantic search / RAG and compared by cosine similarity. Lesson 2
d_model
The dimensionality of the model's vectors — the width every embedding and hidden state shares (e.g. 2048). Lesson 2
Cosine similarity
Similarity of two vectors measured by the angle between them (1 = same direction, 0 = unrelated). The standard ranking metric for sentence-embedding search. Lesson 2
